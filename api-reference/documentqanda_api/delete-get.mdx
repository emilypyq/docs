---
title: Delete Document
api: "GET https://document-query.pyqai.com/delete-document"
description: "This endpoint removes a document from an index."
---

### Query Parameters

<ParamField query="document_source" type="string">
  This is the name of the document you wish to delete.
</ParamField>

<ParamField query="index_name" type="string">
  This is the name of the index you wish to remove the document from.
</ParamField>

<ParamField query="account_id" type="int">
  This is your account ID.
</ParamField>

### Headers

<ParamField header="Authorization" type="string">
  This is your API token associated with your account.
</ParamField>

### Response

<ResponseField name="response" type="string">
  Indicates whether the delete call was successful.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location 'https://document-query.pyqai.com/delete-document?account_id=ExampleAccountID&index_name=ExampleIndexName&document_source=ExampleDocumentSource' \
--header 'Authorization: ExampleToken' \
--data ''
```

</RequestExample>

<ResponseExample>

```json Response
{
    "response": "Successfully deleted document from index"
}
```

</ResponseExample>
