---
title: Compare Documents with Pyq AI
api: "GET https://document-query.pyqai.com/compare-documents"
description: "This endpoint answers comparative questions using 2 documents in the same index."
---

### Query Parameters

<ParamField query="account_id" type="int">
  This is your account ID.
</ParamField>

<ParamField query="index_name" type="string">
  This is the name of the index you wish to query with.
</ParamField>

<ParamField query="input_sequence" type="string">
  This is the query you wish to answer using documents from the given index.
</ParamField>

<ParamField query="document_1" type="string">
  This is the first document you want to consider.
</ParamField>

<ParamField query="document_2" type="string">
  This is the second document you want to consider.
</ParamField>

### Headers

<ParamField header="Authorization" type="string">
  This is your API token associated with your account.
</ParamField>

### Response

<ResponseField name="answer" type="string">
  Pyq AI's response to your query.
</ResponseField>

<ResponseField name="sources" type="list">
  List of sources consulted to get answer.
</ResponseField>

<ResponseField name="source_documents" type="list">
  Most relevant sections from the sources consulted to get the answer.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location 'https://document-query.pyqai.com/compare-documents?account_id=ExampleAccountId&index_name=ExampleIndex&document_1=Document1&document_2=Document2&input_sequence=Which source document contains more location information?' \
--header 'Authorization: ExampleToken' \
--data ''
```

</RequestExample>
