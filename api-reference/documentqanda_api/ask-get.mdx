---
title: Ask Pyq AI
api: "GET https://document-query.pyqai.com/ask"
description: "This endpoint answers a query using documents in an index."
---

### Query Parameters

<ParamField query="account_id" type="int">
  This is your account ID.
</ParamField>

<ParamField query="index_name" type="string">
  This is the name of the index you wish to query with.
</ParamField>

<ParamField query="input_sequence" type="string">
  This is the query you wish to answer using documents from the given index.
</ParamField>

### Headers

<ParamField header="Authorization" type="string">
  This is your API token associated with your account.
</ParamField>

### Response

<ResponseField name="question" type="string">
  Your input_sequence query.
</ResponseField>

<ResponseField name="answer" type="string">
  Pyq AI's response to your query.
</ResponseField>

<ResponseField name="sources" type="list">
  List of sources consulted to get answer.
</ResponseField>

<ResponseField name="source_documents" type="list">
  Most relevant sections from the sources consulted to get the answer.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location 'https://document-query.pyqai.com/ask?account_id=ExampleAccountId&index_name=ExampleIndex&input_sequenceExampleQuery' \
--header 'Authorization: ExampleToken' \
--data ''
```

</RequestExample>
