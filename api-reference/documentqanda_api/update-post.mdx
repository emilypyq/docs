---
title: Update Document
api: "POST https://document-query.pyqai.com/update"
contentType: "multipart/form-data; boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW'"
description: "This endpoint updates a document in an index."
---

### Form Data

<ParamField body="input_sequence" type="file">
  This is the document you want to update. Currently supported file types are md, pdf, and txt.
</ParamField>

<ParamField body="document_source" type="string">
  This is the name of the document you wish to update.
</ParamField>

<ParamField body="index_name" type="string">
  This is the name of the index that contains the document you wish to update.
</ParamField>

<ParamField body="account_id" type="int">
  This is your account ID.
</ParamField>

### Headers

<ParamField header="Authorization" type="string">
  This is your API token associated with your account.
</ParamField>

### Response

<ResponseField name="response" type="string">
   Indicates whether the update document call was successful.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location 'https://document-query.pyqai.com/update' \
--header 'Authorization: ExampleToken' \
--form 'account_id="1"' \
--form 'index_name="Example New Index Name"' \
--form 'document_source="Example 1"' \
--form 'input_sequence=@"example_file_path.txt"'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "response" : "successfully updated document."
}
```

</ResponseExample>
