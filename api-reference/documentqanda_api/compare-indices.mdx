---
title: Compare Indices with Pyq AI
api: "POST https://document-query.pyqai.com/compare-indices"
description: "This endpoint answers comparative questions using 2 indexes."
---

### Body Parameters

<ParamField body="account_id" type="int" required>
  This is your account ID.
</ParamField>

<ParamField body="input_sequence" type="string" required>
  This is the query you wish to answer using documents from the given index.
</ParamField>

<ParamField body="index_name_1" type="string" required>
  This is the first index you want to consider.
</ParamField>

<ParamField body="index_name_2" type="string" required>
  This is the second index you want to consider.
</ParamField>

### Headers

<ParamField header="Authorization" type="string" required>
  This is your API token associated with your account.
</ParamField>

### Response

<ResponseField name="answer" type="string">
  Pyq AI's response to your query.
</ResponseField>

<ResponseField name="sources" type="list">
  List of sources consulted to get answer.
</ResponseField>

<ResponseField name="source_documents" type="list">
  Most relevant sections from the sources consulted to get the answer.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location 'https://document-query.pyqai.com/compare-indices' \
--header 'Authorization: ExampleAPIToken' \
--header 'Content-Type: application/json' \
--data '{"account_id" : ExampleAccountID, "index_name_1" : "ExampleIndex1", "index_name_2" : "ExampleIndex3", "input_sequence" : "Example comparative query"}'
```

</RequestExample>
