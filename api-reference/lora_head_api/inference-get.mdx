### Query Parameters

<ParamField query="account_id" type="int">
  This is your account ID.
</ParamField>

<ParamField query="input" type="string">
  This is the input for your specified adapter.
</ParamField>

<ParamField query="adapter_name" type="string">
  This is the  name of the adapter you want to run inference on.
</ParamField>

### Headers

<ParamField header="Authorization" type="string">
  This is your API token associated with your account.
</ParamField>

### Response

<ResponseField name="response" type="json">
  The output of your adapter.
</ResponseField>
