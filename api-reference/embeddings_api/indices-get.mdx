---
title: List Indices
api: "GET https://api.mintlify.com/indices"
description: "This endpoint lists all indices associated with your account."
---

### Body

<ParamField body="account_id" type="int">
  This is your account ID.
</ParamField>

<ParamField header="Authorization" type="string">
  This is your API token associated with your account.
</ParamField>

### Response

<ResponseField name="response" type="list">
  List of all indicies associated with your account ID.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location 'https://document-query.pyqai.com/indices?account_id=ExampleAccountID' \
--header 'Authorization: ExampleToken' \
--data ''
```

</RequestExample>

<ResponseExample>

```json Response
{
    "response": [
        "Index1",
        "Index2",
        "Index3"
    ]
}
```

</ResponseExample>
