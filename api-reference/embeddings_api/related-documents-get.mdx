title: Related Documents

### Body

<ParamField body="account_id" type="int">
  This is your account ID.
</ParamField>

<ParamField body="index_name" type="string">
  This is the name of the index you want to search for the specifed document.
</ParamField>

<ParamField body="input_sequence" type="string">
  This is the query we use to search for related documents.
</ParamField>

<ParamField body="num_docs" type="int">
  Number of related documents you want to receive.
</ParamField>

<ParamField header="Authorization" type="string">
  This is your API token associated with your account.
</ParamField>
