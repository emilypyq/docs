### Form Data

<ParamField body="input_sequence" type="file">
  This is the document you want to update.
</ParamField>

<ParamField body="document_source" type="string">
  This is the name of the document you wish to update.
</ParamField>

<ParamField body="index_name" type="string">
  This is the name of the index that contains the document you wish to update.
</ParamField>

<ParamField body="account_id" type="int">
  This is your account ID.
</ParamField>

### Headers

<ParamField header="Authorization" type="string">
  This is your API token associated with your account.
</ParamField>

### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 1 if successful, 0 if not.
</ResponseField>
