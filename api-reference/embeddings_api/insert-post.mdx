---
title: Insert Document
api: "POST  https://document-query.pyqai.com/insert"
description: "This endpoint inserts a document into an index."
---

### Form Data

<ParamField path="input_sequence" type="file">
  This is the document you want to insert into your index.
</ParamField>

<ParamField path="document_source" type="string">
  This is the name of the document you wish to insert.
</ParamField>

<ParamField path="index_name" type="string">
  This is the name of the index you wish to insert the document into.
</ParamField>

<ParamField path="account_id" type="int">
  This is your account ID.
</ParamField>

### Headers

<ParamField header="Authorization" type="string">
  This is your API token associated with your account.
</ParamField>

### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 1 if successful, 0 if not.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location 'https://document-query.pyqai.com/insert' \
--header 'Authorization: ExampleToken' \
--form 'account_id="1"' \
--form 'index_name="Example New Index Name"' \
--form 'document_source="Example 1"' \
--form 'input_sequence=@"example_file_path.txt"'
```

</RequestExample>

<ResponseExample>

```json Response
{
    "response": "Successfully inserted new document to index"
}
```

</ResponseExample>
