---
title: List Documents
api: "GET https://document-query.pyqai.com/list-documents"
description: "This endpoint lists all document in an index."
contentType: "application/json"
---

### Body

<ParamField query="index_name" type="string">
  This is the name of the index you wish to remove the document from.
</ParamField>

<ParamField query="account_id" type="int">
  This is your account ID.
</ParamField>

<ParamField header="Authorization" type="string">
  This is your API token associated with your account.
</ParamField>

### Response

<ResponseField name="response" type="list">
  List of all documents inside the specified index.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location 'https://document-query.pyqai.com/list-documents?account_id=ExampleIndexId&index_name=ExampleIndexName&input_sequence=ExampleQuery&document_source=ExampleDocumentSource' \
--header 'Authorization: ExampleToken' \
--data ''
```

</RequestExample>

<ResponseExample>

```json Response
{
    "response": [
        "DocumentSource1",
        "DocumentSource2",
        "DocumentSource3",
        "DocumentSource4"
    ]
}
```

</ResponseExample>
